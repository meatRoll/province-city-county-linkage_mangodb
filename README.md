# 中国省市县三级数据库：

## 1. 注意：

* 此插件用于导入省市县三级数据库；
* 请保证您所使用的数据库为mongodb；
* 在请用本客户端前请确保数据库处于链接状态；

## 2. 使用步骤：

* 在导入文件夹中使用`npm start`指令启动；

* 当出现以下提示时请输入服务器地址：
  - ![host](./images/host.png) 
  - 输入enter跳过，默认：localhost

* 当出现以下提示时请输入端口：
  - ![port](./images/port.png) 
  - 输入enter跳过，默认：27017

* 当出现以下提示时请输入数据库名称：

  - ![database](./images/database.png)
